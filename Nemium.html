    Conceptos esenciales del Desarrollo Web


    1. Qué es HTML

    Hiper Text Markup Language o Lenguaje de marcas de Hipertexto. Es el componente más básico de la web. Define el significado y la estructura del contenido

    2. Etiquetas

    Las etiquetas HTML van a atudarnos a brindar una estructura y semántica al contenido de nuestro website y cada una tiene características y usos deiferentes aunque visualmente den un resultado similar.


    Muchas etiquetas son fácil de diferenciar como los headings o las imágenes t otras diferentes como geader, footer, section o div, aquí es donde la semántica será la respuesta y si sabemos un poco de inglés también podremos darnos una pequeña idea de su uso pero en este curso vamos a usar las más importantes y que serán parte de tu día a día como Software Developer.

    3. DOM

    El document object Model es una estructura de árvol que representara todos nuestros proyectos wev como su un árbol genealógico fuera: padres, hijos, hermanos con niveles infinitos. Cuando entendemos esta anidación podemos identificar dependencias, herencias en CSS y que tan complejo es nuestro proyecto

    4. Semántica

    La semántica le brinda sentido a cada elemento existen en un sitio web, algunos elementos será meramente decoretivos y no deben significar nada pero otros serán títulos y deberan estar en diferente jerarquías o alguna secciones serán mas relevantes que ptras, esto ayudará a los motores de búsqueda como Google o Duck Duck Go a diferenciar tu contenido, categorizar y será la herramienta más valiosa para estar en los primeros resultados de búsquedas SEO. También ayudará a la accesibilidad de tu sitio web, para que personas con habilidades diferentes puedan entender cada contenido

    5. Atributos

    Los atributos le dan características extra a las etiquetas para complementar información y son pre programadas por el navegador, algunos atributos funcionan en todas las etiquetas de la misma forma como 'class' pero otros atributos funcionan en solo alguna etiquetas como src

    6. CSS

    Hojas de Estilo en Cascada(del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizando para describir la presentacion de documentos HTML
